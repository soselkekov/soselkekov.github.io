What is Pulpo?

Pulpo is a decentralized cloud gaming platform that allows its users play games and earn coins. It means all gamers can play any game they wish, on any device connected to the Internet, using a decentralized cloud model.

Pulpo will provide a decentralized ecosystem based on blockchain technology which will open a new way to earn money for GPU owners.

This way in perspective may outperform traditional mining by its efficiency and stability.

How it work

Pulpo is a decentralized system. So the process of playing seems as follow way: gamers need to install client app where they can play and make all actions with token.

All computations will execute on remote computers which will be provided by miners so games will be stored and run on their computers/servers. Video results will be streamed directly to client devices.

We plan to develop client apps for such platforms as Windows, Android and IOS.

Miners also have an application where they can customize smart-contract conditions for earning tokens.

Pulpo Token

Pulpo token is based on Ethereum blockchain according with ERC20 standard.

Gamers can use token for paying miners for time they play. Miners in its turn can hold tokens or sell it on crypto-exchanges.
